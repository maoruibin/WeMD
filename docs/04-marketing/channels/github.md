## 📋 Issues 管理

### 响应时间

| 优先级 | 响应时间 |
|--------|----------|
| P0 致命 | 4小时 |
| P1 严重 | 24小时 |
| P2 一般 | 3天 |
| P3 轻微 | 1周 |

### Issue 标签

| 标签 | 用途 |
|------|------|
| bug | Bug 报告 |
| enhancement | 功能建议 |
| documentation | 文档问题 |
| good first issue | 适合新手 |
