# WeMD 产品路线图

> 本文档规划 WeMD 产品的未来发展方向和版本计划

---

## 🎯 产品愿景

打造最专业的 **微信公众号 Markdown 编辑器**，让创作者专注于内容创作，无需担心排版问题。

---

## 🗺️ 版本规划

### v1.0.0 - MVP 版本 ✅

**发布时间**: 2024.Q1

**核心功能**:
- ✅ Markdown 编辑器 (CodeMirror 6)
- ✅ 实时预览
- ✅ 17+ 内置主题
- ✅ 微信深色模式预览
- ✅ 一键复制到微信
- ✅ 本地存储 (IndexedDB)
- ✅ 文件系统访问

---

### v1.1.0 - 协作增强 🚧

**预计发布**: 2024.Q2

**计划功能**:
- [ ] 图片拖拽上传
- [ ] 图片粘贴上传
- [ ] 支持多个图床
- [ ] 图片压缩优化
- [ ] 历史记录增强
- [ ] 导出为 Markdown 文件

**关联文档**:
- `versions/v1.1.0/features/`

---

### v1.2.0 - 编辑体验升级 📋

**预计发布**: 2024.Q3

**计划功能**:
- [ ] 快捷键系统
- [ ] 命令面板 (Cmd+P)
- [ ] 代码块语言高亮

- [ ] 脚注支持
- [ ] 自定义 CSS 主题编辑器

---

### v1.3.0 - 多端同步 🔄

**预计发布**: 2024.Q4

**计划功能**:

---

### v2.0.0 - 团队协作 👥

**预计发布**: 2025.Q1
