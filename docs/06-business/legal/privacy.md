# 隐私政策

> WeMD 如何保护用户隐私

---

## 📋 隐私原则

**WeMD 本地优先，数据不上传**

- 用户内容完全存储在本地
- 不收集用户写作内容
- 不向第三方出售用户数据

---

## 📊 数据收集

### 我们收集什么

| 数据类型 | 用途 | 是否必需 |
|----------|------|----------|
| 本地存储内容 | 编辑器功能 | 是 |
| 用户设置偏好 | 个性化体验 | 是 |
| 匿名使用统计 | 改进产品 | 否 |

### 我们不收集

- 写作内容
- 个人身份信息
- 键盘输入记录
- 浏览历史

---

## 🔒 数据存储

### 本地存储

- IndexedDB: 浏览器本地数据库
- LocalStorage: 用户设置
- File System API: 用户授权的本地文件

### 云端同步 (可选)

- 用户主动开启
- 加密存储
- 可随时删除

---

## 🍪 Cookie 使用

- 仅用于必要的会话管理
- 不用于追踪和广告
