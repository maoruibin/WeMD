## 🎨 设计说明

### 无 UI 变更

本功能为 Bug 修复，不涉及界面变更。

### 现有 UI 保持不变

- 主题选择下拉菜单位置和行为保持不变
- 主题预览效果保持不变
- 主题编辑功能保持不变

---

## ✅ 交互验证点

用户可通过以下操作验证功能修复：

1. **选择主题** → 在主题下拉菜单中选择任意主题
2. **刷新页面** → 按 F5 或点击刷新按钮
3. **验证主题** → 检查主题是否保持为上次选择的主题
4. **重启浏览器** → 关闭后重新打开浏览器
5. **验证主题** → 检查主题是否保持为上次选择的主题

---

## 🎯 预期效果

修复后的行为：

| 操作 | 修复前 | 修复后 |
|------|--------|--------|
| 选择主题后刷新 | 恢复为默认主题 | 保持选中的主题 |
| 选择主题后重新打开 | 恢复为默认主题 | 保持选中的主题 |
| 删除当前使用的自定义主题 | - | 恢复为默认主题 |
