# 竞品分析

> 了解市场，找到差异化机会

---

## 🏆 主要竞品

### 1. Markdown Nice

| 维度 | WeMD | Markdown Nice |
|------|------|---------------|
| 定位 | 公众号专用编辑器 | 通用 Markdown 编辑器 |
| 深色预览 | ✅ 98%+ 还原度 | ❌ 不支持 |
| 本地优先 | ✅ 是 | ❌ 需登录 |
| 开源 | ✅ 完全开源 | ❌ 部分开源 |
| 主题 | 17+ 内置 | 30+ 主题 |

**WeMD 优势**：
- 深色模式预览（独家）
- 本地优先，隐私安全
- 完全开源，可自部署

**WeMD 劣势**：
- 主题数量较少
- 社区活跃度较低

---

### 2. MdNice

| 维度 | WeMD | MdNice |
|------|------|--------|
| 定位 | 公众号专用编辑器 | 通用排版工具 |
| 使用方式 | 本地应用 | Web 应用 |
| 数据存储 | 本地 | 云端 |
| 协作功能 | 规划中 | 支持协作 |

**WeMD 优势**：
- 本地存储，隐私保护
- 无需登录，打开即用
- 深色模式预览

**WeMD 劣势**：
- 无协作功能
- 无云端备份

---

### 3. 微信公众号后台编辑器

| 维度 | WeMD | 微信编辑器 |
|------|------|-----------|
| 编辑方式 | Markdown | 富文本 |
| 深色预览 | ✅ 支持 | ❌ 不支持 |
| 学习成本 | 低（Markdown） | 中（富文本） |
| 样式控制 | 高（主题模板） | 中（样式编辑器） |

**WeMD 优势**：
- Markdown 写作更高效
- 深色模式预览
- 样式一致性更好

**WeMD 劣势**：
- 需要 Markdown 语法知识
- 不支持直接发布

## 🎯 差异化策略

### 核心差异点

| 差异点 | 说明 |
|--------|------|
| **深色模式预览** | 市面独家，基于微信官方算法 |
| **本地优先** | 数据安全，隐私保护 |
| **完全开源** | 可自部署，可定制 |

### 避开红海

| 红海领域 | WeMD 策略 |
|----------|----------|
| 多平台发布 | 专注微信，做深做透 |
| 协作功能 | 本地优先，个人用户为主 |
| AI 写作 | 观望，专注编辑体验 |
