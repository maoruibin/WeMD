## 💡 想法池

### 待评估

| 想法 | 优先级 | 备注 |
|------|--------|------|
| 支持飞书文档导入 | P2 | 用户需求较少 |
| 支持从 URL 导入文章 | P3 | 版权问题需考虑 |
| 浏览器插件 | P1 | 可快速预览 |
| VS Code 插件 | P2 | 开发者需求 |
| 多平台发布（知乎、掘金） | P1 | 扩大用户群 |

### 已拒绝

| 想法 | 拒绝原因 |
|------|----------|
| 富文本编辑模式 | 不符合产品定位 |
| 支持所有平台编辑 | 专注微信 |
| 强制登录才能使用 | 违背本地优先 |
