# WeMD 产品愿景

> 定义产品的存在意义和长期方向

---

## 🎯 存在意义

**WeMD 让公众号创作者专注于内容创作，而非繁琐的排版。**

微信公众号编辑器的不便之处：
- 排版耗时，格式容易错乱
- 复制粘贴样式丢失
- 深色模式效果难以预览
- 无法本地保存，担心内容丢失

WeMD 的解决方案：
- Markdown 写作，专注于内容
- 98%+ 还原度的深色模式预览
- 本地优先，数据安全
- 一键复制，完美兼容

## 🎯 产品定位

### 目标用户

**主要用户**：公众号内容创作者
- 自媒体作者
- 企业运营人员
- 独立写作者

**次要用户**：其他需要 Markdown 转 HTML 的用户
- 技术博客作者
- 文档撰写者

### 使用场景

| 场景 | 用户 | 需求 |
|------|------|------|
| 日常写作 | 自媒体 | 每天写文章，需要高效排版 |
| 深色适配 | 运营 | 确保文章在微信深色模式下正常显示 |
| 多端同步 | 写作者 | 家里和公司都能继续写作 |
| 团队协作 | 企业 | 多人共同维护公众号 |

---

## ❌ 不做什么

### 做减法的原则

| 不做 | 原因 |
|------|------|
| 富文本编辑器 | 违背 Markdown 简洁理念 |
| 支持所有平台 | 专注微信生态，做深做透 |
| 在线协作 | 本地优先，保护隐私 |
| 复杂的插件系统 | 保持轻量，避免臃肿 |
